★ Prox.jl. package to compute the proximal operator of several functions commonly used in nonsmooth optimization problems. Useful as building block to implement large scale optimization algorithms such as ADMM.

★ ForBES. MATLAB solver for nonsmooth optimization, contains a library of mathematical functions to formulate problems arising in control, machine learning, image and signal processing.

★ libForBES. C++ framework for modeling and solving large-scale nonsmooth optimization problems, will allow to interface many high-level languages (including R, Python, Julia) to a unique solver capable of addressing nonsmooth optimization problems from several application fields.

★ libLBFGS. C library providing the structures and routines to implement the limited-memory BFGS algorithm (L-BFGS) for large-scale smooth unconstrained optimization. Contains a Mex interface to MATLAB.
